# Nano Sample Taskpack v0.2

This taskpack is transparent and public. It exists for local smoke tests,
examples, docs, and verifier integration checks. It is not a hidden benchmark.

Files:

- `taskpack.json`: taskpack metadata.
- `public_prompts.jsonl`: agent-visible prompts.
- `verifier_manifest.jsonl`: public verifier rows for local sample scoring.
- `sample_submissions.jsonl`: passing baseline submissions.
- `submissions_good.jsonl`: passing smoke-test submissions.
- `submissions_bad.jsonl`: failing smoke-test submissions.
- `TASK_CARD.md`: taskpack purpose, risk, and evaluation notes.
- `LICENSE.md`: sample taskpack license terms.
