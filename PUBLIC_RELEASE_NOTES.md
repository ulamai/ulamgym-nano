# Public Release Notes

UlamGym Nano v0.2 is meant to be safe to use as a public GitHub repo.

Public by design:

- package source code
- docs and examples
- tests and CI
- transparent sample tasks
- sample verifier manifests for local demos
- public roadmap, citation metadata, and security reporting guidance

Not public by design:

- official hidden challenge manifests
- private task IDs and answer keys
- model/team private submissions
- audit replays
- private reviewer notes
- unreleased problem packs

For official competitions, publish prompts and run scoring through a hosted
verifier, or release only after the challenge is over.
