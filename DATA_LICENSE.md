# Data License for Bundled Sample Tasks

The code in this repository is released under the MIT license in `LICENSE`.
The bundled files under `data/sample_tasks/` are public demo tasks provided for
integration testing, local scoring, and competition prototyping.

Unless a task pack says otherwise in its own `DATA_LICENSE.md`, the bundled
sample task prompts and sample verifier manifests may be used, copied, and
modified for non-confidential technical testing and public demonstrations, with
attribution to Ulam AI.

These sample tasks are not hidden benchmark holdouts. They are intentionally
compact and transparent. They should not be used for leaderboard claims about
frontier model capability.

If you add tasks from third-party sources, you must add source/license metadata
for each task and comply with the upstream source terms.
